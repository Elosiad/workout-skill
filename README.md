# Workout Skill (Math Problem Dissector)

A highly structured pedagogical framework designed to solve mathematical proofs by outputting a rigorous English proof followed by a deeply structured, motivation-driven "Workout" breakdown in Chinese.

## 🌟 About (关于本项目)
This project contains an Agent skill (`SKILL.md`) that enforces a strict two-phase problem-solving methodology:
1. **Standard Solution (English)**: A formal, publication-ready mathematical proof.
2. **Workout Breakdown (Chinese)**: A reverse-engineered, target-driven breakdown of the proof. It translates abstract mathematical conditions into equivalent, connectable forms and highlights the logical bridges and theorems used.

本项目包含一个专属的 Agent 技能配置文件。它的核心理念是将“冰冷严谨的数学答案”与“动机驱动的推导过程”剥离开来。它不仅提供可以直接写到考卷上的全英文标准解答，还会在随后用全中文详细拆解：
- 如何把大问题拆成小问题？
- 为什么要往这个方向想？
- 泛性质、同构定理等“桥梁”是如何恰好连接已知条件与目标的？

## 🚀 Usage (使用指南)
### Installation
Copy the `SKILL.md` file into your agent's skills directory:
```bash
# Example path for Antigravity skills
~/.gemini/config/plugins/general-tools/skills/workout/SKILL.md
```

### Triggering the Skill
Ask your agent any advanced mathematical question (e.g., representation theory, abstract algebra) and explicitly invoke the skill by appending:
> "Prove that... and use the workout skill."
> "证明... 使用 workout 模式"
